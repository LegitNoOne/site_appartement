En pratique

- Clés : la clé plate ouvre toutes les portes ‘utiles’ : porte d’entrée, boîte aux lettres, porte appartement, garage pour vélos, au -1.

- Chauffage : le thermostat se trouve à droite de la porte d’entrée : si nécessaire, changer la température avec les petites flèches haut/bas. Par défaut, la température affichée est celle de la pièce.

- Electricité : en principe, tout est en fonction : si ce n’est pas le cas, le différentiel se trouve dans le débarras.

- Eau : en principe, le robinet général est ouvert : si ce n’est pas le cas, dans le couloir, à gauche en sortant de l’ascenseur, il y a une trappe : ouvrir le robinet général (position horizontale).

- Poubelles : 4 types de tris (les sacs se trouvent dans le tiroir du bas, à droite de la taque de cuisson), le ramassage se fait aux jours renseignés sur un folder dans le même tiroir : - Sacs bleus : tous les emballages de types bouteilles plastiques, conserves, tetrapack, etc… - Cartons, papiers : pas de sacs prévus, les collectionner dans une boîte à part…. - Verres : il y a des bulles à verres en ville… (pour les courageux) - Sacs bruns : tout le reste, y compris les déchets ménagers.

- TV : abonnement Telenet, qui donne accès à de nombreuses chaînes, notamment francophones (130 et suivantes) ainsi que les radios – liste des chaînes sur le folder à côté de la TV ; lecteur CD, radio FM et home cinéma.

- WIFI : le code Wifi est affiché dans le débarras, près du compteur électrique

- En cas de problème technique dans les communs : contacter le syndic,, Agimobel, dont le n° est renseigné à la porte d’entrée de l’immeuble et dans l’ascenseur – fermé le mercredi et le dimanche.

- Pour toute question : notre GSM est le 00 352 621 202 244.

- Nettoyage en fin de séjour : l’idée est que l’appartement doit se retrouver dans l’état initial. 2 formules sont possibles : soit la prise en charge individuelle du nettoyage, soit le nettoyage (sans présence nécessaire) facturé 60 euros.
